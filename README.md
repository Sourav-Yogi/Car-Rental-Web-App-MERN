# MERN Car Rental Platform

A full-stack car rental web application built using MERN stack.

## Features
- User authentication (JWT)
- Owner dashboard
- Car listing & management
- Booking system
- Image upload (ImageKit)
- Responsive UI

## Tech Stack
- React (Vite)
- Node.js
- Express
- MongoDB
- Tailwind CSS
